DMX Bluetooth Remote

    • Download and install Bluetooth Terminal from Google Play:https://play.google.com/store/apps/details?id=Qwerty.BluetoothTerminal
    • Connect to the HC-05 module using the default password of 1234
    • In the terminal setup, make sure "Append newline (\r\n)" is check marked
    • If you're working with the light, make sure it is on: 4 channel mode , set up on channel 1 and that it is on NASL
    • The following commands are possible:
    • "U" Tilt up slightly
    • "D" Tilt down slightly
    • "L" Pan left slightly
    • "R" Pan right slightly
    • "P###" Pan to setting from 0-255 (Example: type in P20 and hit send)
    • "T###" Tilt to setting from 0-255
    • "C###" Change color to setting from 0-255
    • "B###" Change brightness to setting from 0-20
